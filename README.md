# IsuBreak-GameProject

IsuBreak - Hapishaneden Kaçış Oyunu


Oyun programlama 2 dersimiz için yapmış olduğum oyun projem karşınızda. Oyunumuz Prison Break tarzı hapishane kaçış temalı bir oyundur. Oyunum %90 özgünlük ile "Berke Akar"'a (bana) aittir.


# Kullanılan Teknolojiler:
- Unity (Oyun Motoru)
- C# (Programlama)
- Blender (Oyun İçi 3D Tasarımlar)
- Photoshop (Oyun İçi Görsel Tasarımlar)
